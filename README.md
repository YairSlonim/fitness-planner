# Workout Planner (React + TypeScript)

A personal workout planner project.
Current features:
- General exercises catalog grouped by muscle group
- “My workout” page with selectable day (WIP)
- Exercise cards UI (responsive grid)

## Tech Stack
- React
- TypeScript
- React Router
- CSS

## Getting Started

### Prerequisites
- Node.js (LTS recommended)
- npm
